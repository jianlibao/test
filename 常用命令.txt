1、sudo shutdown  -h  now  // 立刻进行关机

2、sudo shutdown  -r   now  // 重新启动计算机

3、sudo reboot   // 重新启动计算机    

4、ls  -l  // 用于查看所保存文件的详细信息

5、vi编辑器的用法
    第1步：输入vi 后面跟上工程名称
    第2步：输入一个小写的i
    第3步：输入esc键
    第4步：输入: 后面跟上wq(保存退出)或ql(不保存退出)
    第5步：编译程序 输入gcc 后面跟上c语言程序，
    第6步：生成可执行文件后，然后输入./a.out
